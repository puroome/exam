📄 PDF 저장 권장 방법
  1. 한컴오피스 → [파일] → [인쇄] 메뉴 열기
  2. [인쇄 방식] → "기본 인쇄" 선택 (소책자 인쇄는 페이지가 뒤섞여 정확도가 낮습니다)
  3. [프린터 선택] → "PDF 저장" 또는 "Microsoft Print to PDF"
  4. 저장된 세로형 PDF를 위에 업로드
· 동점자는 중간석차백분율(NEIS 표준)로 등급 부여

📄 한컴오피스 PDF 저장 권장 설정

아래 화면처럼 [인쇄 방식 → 기본 인쇄] + [프린터 → PDF 저장] 으로 만들면 인식 정확도가 가장 높습니다.
① 프린터 선택
우측 두 번째의 PDF 저장 아이콘 클릭
② 인쇄 방식
기본 인쇄 라디오 선택 (자동 인쇄 유지)
⚠ 피해야 할 옵션
소책자 모양으로 찍기 · 모아 찍기 2쪽씩 — Exam-Lens 인식 어려움
③ 인쇄(D) 클릭
우측 상단 파란 [인쇄(D)] 버튼 → 저장 위치 선택 → 완료
💡 왜 권장하나요? 소책자(saddle-stitch) 모드로 저장하면 PDF 페이지 순서가 [12,1] [2,11] [10,3] ... 처럼 인쇄용으로 뒤섞입니다. 자동 디코딩 기능이 있긴 하나 한컴 옵션·imposition 방식에 따라 인식 정확도가 떨어집니다. 세로형 1up PDF는 모든 양식에서 안정적으로 동작합니다.
