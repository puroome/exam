배점-정답률 적정성 분석?

출제자가 부여한 배점(난이도 의도)과 실제 정답률의 일치 여부 점검 · 이상적으로는 음의 상관(배점↑ → 정답률↓) · 산점도 점 클릭 시 문항번호 표시

성취도(A~E·I) × 문항 정답률 매트릭스?

상위 등급일수록 정답률이 높아야 정상. 셀 색: 초록=높음 → 빨강=낮음 · 패턴 깨짐은 3그룹(상위 A·B / 중위 C·D / 하위 E·I) 가중 정답률을 기준으로 비교 — 빨간 테두리는 상위 그룹보다 정답률이 더 높은 그룹의 셀 · ★ 검은 테두리는 해당 문항의 판별 성취수준(정답률 65% 이상이 유지되는 가장 낮은 등급 = 변별 경계)

답지 반응 분포 (선지별 응답률)

셀 클릭 시 해당 응답 학생 명단 · 정답 선지 ✓ 초록 · 0% 선지는 매력도 부족 신호 · 30% 이상 매력적 오답은 빨강 · '복수답안' 컬럼은 알파벳 코드 (A=1,2 / B=1,3 …)

출제 의도(NEIS 난이도) vs 실제 정답률?

출제 계획서(NEIS 문항정보표)의 난이도와 실제 학생 응답 결과를 비교 · Spearman 순위 상관으로 출제 의도의 일치도를 단일 지표로 제시

출제 의도와 가장 어긋난 문항 — 의도 대비 정답률 괴리 큰 순

의도 난이도 × 실제 난이도 매트릭스

대각선(왼쪽 위 → 오른쪽 아래)에 채워지면 의도와 실제가 일치. 좌하/우상 셀이 굵게 보일수록 의도와 실제가 어긋난 문항.
